## Responsible Disclosure Philosophy

I believe security research should improve the overall security ecosystem.

When identifying potential vulnerabilities:

1. Verify findings safely and responsibly.
2. Minimize impact during testing.
3. Report directly to the affected organization.
4. Allow adequate remediation time.
5. Publish only non-sensitive information after remediation or approval.
